    El OLED Studio es una herramienta web de matriz LED que reúne varias funciones:
    – editor de píxeles potente
    Optimizado para crear bitmaps e iconos para pantallas OLED de 128×32 (SSD1306)
    – sistema flexible de importación y plantillas
    Conversión directa en arreglos de código para Arduino, ESP32 y otros microcontroladores
    – cinco idiomas, sistema de errores y una variedad de herramientas para un procesamiento rápido
    …integrado en una arquitectura modular y preparada para el futuro.

    El “programa” nació de la necesidad de facilitar el trabajo con microcontroladores, tanto para uno mismo como para otros. Una vista previa rápida o una conversión automatizada ahorra mucho tiempo y evita frustraciones innecesarias. Además, OLED Studio —desarrollado en HTML y JavaScript— sirve como muestra de trabajo para demostrar habilidades técnicas. No es programación de aficionado, sino el resultado de una arquitectura de software profesional, modular y preparada para el futuro.
    Destacados y Funciones
    Funciones del Editor:
    Pixel‑to‑Hex genera matrices C compactas (p. ej. 0xFF). Pixel‑to‑Binary crea representaciones binarias claras (p. ej. 0b10101010). Pixel‑to‑ASCII produce vistas ASCII con caracteres configurables.
    Compatible con varios tamaños de cuadrícula como 8×8, 16×16, 32×32, 8×16 y 16×8.
    Mini vista previa (1:1) para inspección rápida de iconos. Alternar filas o columnas completas con un solo clic. Sello de texto Su entrada de texto se convierte en un icono colocable libremente. Sistema de plantillas y sellos para patrones reutilizables. Edición precisa mediante una estructura de datos interna unificada.

    Funciones del Grid Principal:
    Vista previa para diseños de 8×8, 16×16 y 32×32. Cambio directo entre diferentes vistas de cuadrícula. Visualización clara de la matriz de píxeles activa. Compatibilidad con estructuración vertical y horizontal. Use las teclas de flecha para mover el contenido rápidamente.

    Herramientas del Workshop:
    Espejo horizontal y vertical. Rotación en pasos de 90°. Escalado de patrones (solo en Workshop). Inversión de patrones completos. Movimiento preciso al píxel de iconos y elementos.
    Combinar y recolocar sellos. Edición flexible de elementos individuales o agrupados.

    Sistema de Importación/Exportación:
    Importación de códigos hexadecimales y binarios. Procesamiento correcto de bitmaps SSD1306, incluido el vertical paging. Detección automática de ancho y alto. Conversión de byte a matriz para todos los formatos compatibles. Exportación a matrices hex, patrones binarios y estructuras internas.

    Arquitectura y Lógica del Sistema:
    Separación clara entre lógica, renderizado y gestión. Estructura modular para una escalabilidad a largo plazo. Detección y análisis basados en reglas. Gestión de errores robusta para un procesamiento estable. Datos unificados en todos los módulos.

    Interacción y Flujo de Trabajo:
    Editor intuitivo con retroalimentación visual directa. Funcionamiento completamente offline mediante recursos locales. Selección automática entre 4 idiomas principales alemán, inglés, francés y español sin dependencias externas. Lógica de interacción coherente mediante elementos UI unificados. Estructura clara de paneles para Editor, Workshop e Import. Un selector de temas oculto (Ctrl+K).
